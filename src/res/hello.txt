WTAB v.3.01 23.12.2016
Удаляет из таблицы скриншотов строки с несуществующими файлами .jpg
c помощью Word промежуточно преобразует RTF в DOCX и обратно
c помощью pkzipc формирует архив zip
>java -jar wtab.jar Dir\\File
Dir  - директория, где находятся  файл с таблицей и файлы скриншотов
File - файл Word RTF с таблицей скриншотов (задается как начало имени файла)
По умолчанию - %s
В Dir формируется архив _a.zip с файлами акта и протокола, если изменилось кол-во скриншотов.
По окончанию работы в буфере обмена количество нарушений.
